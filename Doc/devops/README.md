# Liteyuki DevOps 工程文档

本目录存放面向开发、评审、测试和交付的标准工程文档。它和 `docs/` 文档站职责不同：

- `docs/`：面向用户和开发者的在线文档站，强调快速上手和产品使用。
- `Doc/devops/`：面向项目实现和技术评审的工程文档，强调需求、架构、数据模型、API、测试和部署。

| [02-概要设计.md](cn/02-概要设计.md) | 说明系统架构、模块边界、核心流程和依赖 |

| [04-数据库设计.md](cn/04-数据库设计.md) | 说明核心模型、表关系、唯一约束、删除和迁移策略 |
