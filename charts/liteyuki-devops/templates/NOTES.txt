Luna DevOps has been installed.

Check pods:
  kubectl -n {{ .Release.Namespace }} get pods -l app.kubernetes.io/instance={{ .Release.Name }}

Open the console with port-forward:
  kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "liteyuki-devops.apiName" . }} 8088:{{ .Values.api.service.port }}

Then visit:
  {{ .Values.app.publicBaseUrl }}

If you enabled ingress, make sure app.publicBaseUrl matches the public URL.
