# Liteyuki DevOps Docs

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get started

Start the dev server (default: `http://localhost:5274`):

```bash
pnpm run dev
```

Build the website for production:

```bash
pnpm run build
```

Preview the production build locally (default: `http://localhost:5274`):

```bash
pnpm run preview
```
