# 历史文档

早期产品方案、代码健康 SOP 和场景测试记录仍然保留，但不再作为普通用户的操作指南。需要追溯设计背景、历史决策或测试过程时，可以到仓库的 `notes/` 目录查阅。

## 可以查到哪些历史资料

| 文件 | 用途 |
| --- | --- |
| `notes/01-产品与一体化方案.md` | 产品背景、MVP 范围和能力边界。 |
| `notes/07-代码健康检查SOP.md` | 长期协作中的代码健康检查节奏。 |
| `notes/场景模拟.md` | 核心用户场景和验收路径。 |
| `notes/场景测试问题记录.md` | 场景测试发现的问题、修复和复测记录。 |

这些文件仍是项目知识的一部分。已经稳定、适合公开使用的内容会继续整理到文档站；过程记录和历史讨论则保留在 `notes/` 中。
